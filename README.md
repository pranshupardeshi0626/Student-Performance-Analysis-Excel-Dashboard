# ЁЯУК Student Performance Analysis Excel Dashboard

An interactive Microsoft Excel dashboard designed to analyze student academic performance using formulas, Pivot Tables, Pivot Charts, XLOOKUP, Data Validation, Conditional Formatting, and dynamic dashboards.

---

## ЁЯУМ Project Overview

Educational institutions collect large amounts of student performance data, but extracting meaningful insights manually can be difficult.

This project transforms raw academic data into an interactive Excel dashboard that allows users to explore student performance, compare subject scores, monitor GPA, and evaluate attendance and study habits.

---

## ЁЯОп Business Problem

Educational institutions need an easy-to-use reporting solution to:

- Monitor student academic performance
- Identify high and low performers
- Compare subject-wise scores
- Track GPA trends
- Analyze attendance and study time
- Support academic decision-making
